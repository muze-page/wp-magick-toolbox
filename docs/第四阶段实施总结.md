## 四、SQL 安全加固

**变更文件**: `admin/partials/function/download-sql-table.php`

`SELECT * FROM {$searchTableName}` 添加安全注释，说明 `prepare()` 无法参数化表名标识符，当前通过 3 层防护保证安全：
1. `sanitize_text_field()` 输入清理
2. 正则白名单 `/^[a-zA-Z0-9_]+$/` 验证
3. `SHOW TABLES LIKE %s` 存在性检查
