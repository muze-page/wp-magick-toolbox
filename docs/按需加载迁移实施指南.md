## Phase 1：基础设施准备（1-2 天）

### Step 1.1：创建 MaBox_Helpers

**文件**：`includes/class-magick-helpers.php`

```php
<?php
/**
 * 公共工具类
 * 
 * 所有功能模块共享的公共逻辑。
 * 此类在插件初始化时必然加载，模块可安全调用。
 */

if (!class_exists('MaBox_Helpers')) {
    class MaBox_Helpers
    {
        /**
         * 获取用户真实 IP
         */
        public static function get_real_ip()
        {
            $ip = '';
            if (!empty($_SERVER['HTTP_X_FORWARDED_FOR'])) {
                $ips = explode(',', $_SERVER['HTTP_X_FORWARDED_FOR']);
                $ip = trim($ips[0]);
            } elseif (!empty($_SERVER['HTTP_CLIENT_IP'])) {
                $ip = $_SERVER['HTTP_CLIENT_IP'];
            } else {
                $ip = $_SERVER['REMOTE_ADDR'] ?? '';
            }
            return sanitize_text_field($ip);
        }

        /**
         * 判断是否为移动端
         */
        public static function is_mobile()
        {
            return wp_is_mobile();
        }

        /**
         * 判断当前用户是否已登录
         */
        public static function is_logged_in()
        {
            return is_user_logged_in();
        }

        /**
         * 获取当前文章 ID
         */
        public static function get_current_post_id()
        {
            return get_the_ID();
        }

        /**
         * 安全获取配置值（MaBox_Admin::get_config 的快捷方式）
         */
        public static function get_config($config, $key, $default = false)
        {
            return MaBox_Admin::get_config($config, $key, $default);
        }
    }
}
```

### Step 2.3：改造 MaBox_Admin::load()

```php
/**
 * 加载所需的依赖项（按需加载版本）
 */
public function load()
{
    // 获取选项
    $option = get_option(MAGICK_MIXTURE_OPTION);
    if (empty($option)) {
        return;
    }
    
    // 尝试从缓存读取路由表
    $active_modules = false;
    if (function_exists('wp_cache_get')) {
        $active_modules = wp_cache_get('mabox_active_modules', 'mabox');
    }
    
    // 缓存未命中，从数据库读取
    if ($active_modules === false) {
        $active_modules = get_option('Magick_ToolBox_Acttive_Modules', array());
    }
    
    // 降级：如果路由表为空，回退到传统加载模式
    if (empty($active_modules)) {
        $this->load_legacy($option);
        return;
    }
    
    // 按需加载
    $this->load_on_demand($active_modules, $option);
}

/**
 * 传统加载模式（降级回退）
 */
private function load_legacy($option)
{
    require_once plugin_dir_path(__FILE__) . 'partials/optimize/index.php';
    MaBox_Optimize::run();
    
    require_once plugin_dir_path(__FILE__) . 'partials/function/index.php';
    MaBox_Function::run();
    
    require_once plugin_dir_path(__FILE__) . 'partials/h5.php';
    MaBox_H5::run();
    
    require_once plugin_dir_path(__FILE__) . 'partials/login/index.php';
    Npcink_Login::run();
    
    require_once plugin_dir_path(__FILE__) . 'partials/page/index.php';
    Npcink_Page::run();
    
    require_once plugin_dir_path(__FILE__) . 'partials/shortcode/index.php';
    MaBox_ShortCode::run();
    
    require_once plugin_dir_path(__FILE__) . 'partials/template/index.php';
    Npcink_Template::run();
}

/**
 * 按需加载模式
 */
private function load_on_demand($active_modules, $option)
{
    foreach ($active_modules as $module_path) {
        $file = plugin_dir_path(__FILE__) . 'partials/' . $module_path;
        if (file_exists($file)) {
            require_once $file;
            
            // 根据模块路径调用对应的 run/activate 方法
            $this->activate_module($module_path, $option);
        }
    }
}
```

### Step 3.3：仪表盘组件

```tsx
import React from 'react';
import { Card, Row, Col, Button, Tag, message } from 'antd';
import { RocketOutlined, SparklesOutlined, DollarOutlined } from '@ant-design/icons';
import { presets } from '../tool/presets';
import { useContext } from 'react';
import { DataContext } from '../context/DataContext';

const Dashboard: React.FC = () => {
  const { optionData, setOptionData } = useContext(DataContext);
  
  // 计算已开启功能数
  const countEnabled = (obj: any): number => {
    let count = 0;
    for (const key in obj) {
      if (typeof obj[key] === 'object' && obj[key] !== null) {
        count += countEnabled(obj[key]);
      } else if (obj[key] === true || (typeof obj[key] === 'string' && obj[key] !== 'false')) {
        count++;
      }
    }
    return count;
  };
  
  const totalFeatures = 90;
  const enabledFeatures = countEnabled(optionData);
  
  const applyPreset = (presetId: string) => {
    const preset = presets.find(p => p.id === presetId);
    if (preset) {
      // 合并预设配置
      const newConfig = { ...optionData, ...preset.config };
      setOptionData(newConfig);
      message.success(`已应用「${preset.name}」预设`);
    }
  };
  
  return (
    <div style={{ padding: '24px' }}>
      <h2>WP Magick Toolbox 仪表盘</h2>
      
      {/* 运行状态 */}
      <Card title="运行状态" style={{ marginBottom: 24 }}>
        <Row gutter={16}>
          <Col span={8}>
            <div style={{ textAlign: 'center' }}>
              <div style={{ fontSize: 32, fontWeight: 'bold' }}>{totalFeatures}</div>
              <div style={{ color: '#666' }}>可用功能</div>
            </div>
          </Col>
          <Col span={8}>
            <div style={{ textAlign: 'center' }}>
              <div style={{ fontSize: 32, fontWeight: 'bold', color: '#52c41a' }}>
                {enabledFeatures}
              </div>
              <div style={{ color: '#666' }}>已开启</div>
            </div>
          </Col>
          <Col span={8}>
            <div style={{ textAlign: 'center' }}>
              <div style={{ fontSize: 32, fontWeight: 'bold', color: '#1890ff' }}>
                {totalFeatures - enabledFeatures}
              </div>
              <div style={{ color: '#666' }}>已关闭</div>
            </div>
          </Col>
        </Row>
      </Card>
      
      {/* 预设套件 */}
      <Card title="一键预设套件">
        <Row gutter={16}>
          {presets.map(preset => (
            <Col span={8} key={preset.id}>
              <Card
                hoverable
                actions={[
                  <Button 
                    type="primary" 
                    icon={
                      preset.id === 'pure' ? <RocketOutlined /> :
                      preset.id === 'fancy' ? <SparklesOutlined /> :
                      <DollarOutlined />
                    }
                    onClick={() => applyPreset(preset.id)}
                  >
                    应用此预设
                  </Button>
                ]}
              >
                <Card.Meta
                  title={preset.name}
                  description={preset.description}
                />
              </Card>
            </Col>
          ))}
        </Row>
      </Card>
    </div>
  );
};

export default Dashboard;
```

## Phase 4：静态资源优化（2-3 天）

### Step 4.1：资源分级审查

遍历所有模块文件，按以下标准分类：

| 标准 | 级别 | 处理方式 |
|---|---|---|
| 代码量 < 2KB | L1 内联 | 直接 `echo` 输出 |
| 代码量 2-20KB | L2 独立 | `wp_enqueue_script` 入队 |
| 代码量 > 20KB 或有第三方依赖 | L3 延迟 | 条件入队 + `defer` |

### Step 4.2：改造示例

**改造前**（直接输出）：

```php
public static function render()
{
    echo '<script src="' . plugin_dir_url(__DIR__) . 'public/js/fish.js"></script>';
}
```

**改造后**（条件入队）：

```php
public static function render()
{
    wp_enqueue_script(
        'mabox-bottom-fish',
        plugin_dir_url(__DIR__) . 'public/dist/bottom-fish.js',
        array(),
        MAGICK_MIXTURE_VERSION,
        true
    );
    wp_add_inline_script(
        'mabox-bottom-fish',
        'FishConfig.init(' . json_encode(self::$option) . ');'
    );
}
```

## 回滚方案

如果按需加载架构在生产环境出现问题：

1. **快速回滚**：将 `Magick_ToolBox_Acttive_Modules` 选项设为空数组
   ```php
   update_option('Magick_ToolBox_Acttive_Modules', array());
   ```
   系统将自动回退到 `load_legacy()` 传统加载模式。

2. **代码回滚**：恢复 `MaBox_Admin::load()` 为改造前的版本，保留路由表生成逻辑以便后续重新启用。

3. **数据保留**：路由表数据保留在数据库中，修复问题后可重新启用。
