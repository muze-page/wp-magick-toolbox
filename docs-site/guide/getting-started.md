# 安装与使用

## 环境要求

- WordPress 4.6 或更高版本
- PHP 7.4 或更高版本
- 支持现代浏览器（Chrome、Firefox、Safari、Edge）

## 安装步骤

### 方式一：后台上传（推荐）

1. 从 [Gitee](https://gitee.com/gitgreat/wp-magick-toolbox) 下载最新版本的 ZIP 包
2. 登录 WordPress 后台，进入 **插件 → 安装插件**
3. 点击 **上传插件**，选择下载的 ZIP 文件
4. 点击 **现在安装**，等待安装完成
5. 点击 **启用插件**

### 方式二：FTP 上传

1. 解压 ZIP 包，得到 `wp-magick-toolbox` 文件夹
2. 通过 FTP 将文件夹上传至 `wp-content/plugins/` 目录
3. 在 WordPress 后台 **插件** 页面找到并启用

## 初始配置

启用插件后：

1. 在左侧菜单找到 **魔法工具箱**
2. 进入设置页面后，点击一次 **保存** 按钮
3. 此步骤用于初始化配置数据，确保所有功能正常加载

## 使用仪表盘

进入设置页面后，默认显示 **仪表盘** Tab：

- **站点健康评分**：根据已启用功能计算的综合评分
- **已启用功能**：显示当前开启的功能数量
- **建议开启**：系统推荐的安全/实用功能
- **建议谨慎**：可能影响性能或体验的功能提示
- **安全状态**：登录保护、评论防护、SEO 优化等状态概览

## 一键配置方案

如果不确定该开启哪些功能，可使用内置的一键配置方案：

1. 进入 **一键配置** Tab
2. 选择适合您站点的方案（如「个人博客推荐配置」）
3. 点击 **应用此配置**，确认后系统自动批量开启/关闭功能

> 应用前会提示覆盖现有配置，建议先使用 **配置备份** 功能保存当前状态。

## 配置备份

插件自动保留最近 5 次配置快照：

1. 进入 **配置备份** Tab
2. 查看快照列表，点击 **恢复** 可回滚到指定版本
3. 也可手动导出当前配置为 JSON 文件，用于迁移或备份

## 常见问题

### 启用后站点白屏怎么办？

插件采用按需加载机制，未启用的功能不会执行。如果启用后出现异常：
1. 进入 WordPress 后台 → 插件 → 停用该插件
2. 检查 PHP 错误日志
3. 确认 PHP 版本 ≥ 7.4

### 如何导出/导入配置？

在 **配置备份** Tab 中：
- **导出**：点击导出按钮，下载 JSON 配置文件
- **导入**：点击导入按钮，选择 JSON 文件，系统会自动校验并导入

### 设置页面加载慢？

设置页面使用 React 构建，首次加载需要下载 JS/CSS 资源。后续访问会使用浏览器缓存。确保您的站点没有启用过度激进的缓存插件。
