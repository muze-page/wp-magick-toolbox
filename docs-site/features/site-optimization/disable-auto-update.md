# 禁用自动更新

::: warning 谨慎
关闭 WordPress 自动更新可能带来安全风险，建议配合手动更新策略使用。
:::

## 功能说明

WordPress 默认会定期检查核心、插件和主题的更新，并在后台提示更新。此功能关闭所有自动更新检查。

## 使用场景

- 生产环境：避免自动更新导致兼容性问题
- 定制站点：已深度定制主题/插件，更新可能覆盖修改
- 测试环境：保持版本一致以便复现问题

## 开启方法

1. 进入 **魔法工具箱** → **站点优化**
2. 开启「禁用自动更新」
3. 保存配置

## 注意事项

关闭后需手动检查 WordPress 核心、插件和主题的安全更新，建议每月检查一次。
