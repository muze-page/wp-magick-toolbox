# 禁止 Title 转义

::: tip 推荐
安全且建议开启。不影响站点性能。
:::

## 功能说明

WordPress 默认会将标题中的半角连字符 `-` 自动转义为全角破折号 `—`（em dash）。此功能关闭该行为，保持标题中的原始字符。

## 使用场景

- 技术博客：标题中包含版本号如 `v2.0-rc1`，不希望被转义
- 英文站：保持英文标题中的连字符原样显示
- SEO 需求：确保 URL slug 与标题一致

## 开启方法

1. 进入 **魔法工具箱** 设置页
2. 切换到 **站点优化** Tab
3. 找到「禁止 Title 转义」，开启开关
4. 点击页面底部 **保存** 按钮

## 注意事项

开启后仅影响新发布文章的标题显示，不影响已有文章。
