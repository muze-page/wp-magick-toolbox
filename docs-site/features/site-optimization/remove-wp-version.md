# 移除 WP 版本号

::: tip 推荐
安全且建议开启。减少站点信息暴露。
:::

## 功能说明

WordPress 默认在前端 HTML 和 RSS 订阅中输出当前版本号。此功能移除这些信息，减少攻击者获取版本信息的机会。

## 使用场景

- 安全加固：避免暴露 WordPress 版本，降低被针对性攻击的风险
- SEO 优化：减少 HTML 中的冗余 meta 标签

## 开启方法

1. 进入 **魔法工具箱** → **站点优化**
2. 开启「移除 WP 版本号」
3. 保存配置

## 注意事项

此功能仅移除前端和 RSS 中的版本号，不影响后台显示和升级功能。
