# 分类链接去 category

::: tip 推荐
安全且建议开启。优化 URL 结构。
:::

## 功能说明

WordPress 默认分类 URL 包含 `/category/` 前缀（如 `example.com/category/news`）。此功能移除该前缀，使 URL 更简洁（如 `example.com/news`）。

## 使用场景

- SEO 优化：更短的 URL 有利于搜索引擎收录
- 用户体验：URL 更简洁易记
- 品牌一致性：去除 WordPress 默认标记

## 开启方法

1. 进入 **魔法工具箱** → **站点优化**
2. 开启「分类链接去 category」
3. 保存配置

## 注意事项

开启后需更新已发布的分类链接，建议配置 301 重定向避免旧链接 404。
