# AI 审核引擎

::: tip 推荐
内容安全建议开启。
:::

## 功能说明

使用 AI 对评论进行安全审核，自动识别广告、灌水和敏感内容。支持多 Provider 自动降级。

## Provider 支持

- **DeepSeek**（默认，性价比高）
- **阿里云内容安全**（企业用户偏好）
- **自定义 API**（用户自行配置 endpoint）
- **本地规则引擎**（无 API 时自动降级）

## 开启方法

1. 进入 **魔法工具箱** → **其他功能**
2. 开启「AI 审核引擎」
3. 选择 Provider 并配置 API Key
4. 选择审核模式（严格/宽松）
5. 保存配置

## 注意事项

无 API 配置时自动降级到本地规则引擎（关键词 + 正则），仍可提供基础防护。
