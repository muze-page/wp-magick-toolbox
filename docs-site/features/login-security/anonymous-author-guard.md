# 限制匿名作者枚举

该能力收紧匿名请求通过数字作者参数和 WordPress REST 用户接口获取作者信息的范围，同时保留已授权编辑工作流。

## 开启方法

1. 进入 **魔法工具箱** → **国内生态** → **登录安全**。
2. 开启“限制匿名作者枚举”。
3. 保存配置。

该能力对应服务端字段：

```text
anonymous_author_guard_enabled
```

## 运行边界

- 匿名数字作者查询会被收紧。
- 匿名 REST 用户读取会被限制。
- 已登录且具备相应权限的编辑器、管理员和 REST 请求仍可读取完成工作所需的作者数据。
- 该能力可以独立启用，不依赖登录尝试保护。

::: info 不是身份隐藏保证
文章署名、作者归档、主题模板或其他插件仍可能公开作者显示名称。这项设置减少常见的匿名枚举入口，但不承诺隐藏所有作者信息，也不替代账号密码和权限管理。
:::

## 验证建议

保存后分别检查：

1. 无痕窗口中的数字作者查询和 REST 用户读取受到限制。
2. 已登录编辑器仍能在文章编辑器中选择作者。
3. 管理员和已授权 REST 客户端不受匿名规则误伤。
