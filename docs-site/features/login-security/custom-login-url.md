# 自定义登录入口

::: warning 谨慎
开启后需记住新登录地址。
:::

::: warning 当前依赖
登录安全运行时目前以“账号失败限制”为统一加载门槛。仅开启自定义登录入口不会单独加载该模块。
:::

## 功能说明

将默认的 `wp-login.php` 登录地址修改为自定义路径，原地址自动跳转 404。

## 开启方法

1. 进入 **魔法工具箱** → **国内生态** → **登录安全**
2. 开启「自定义登录入口」
3. 设置自定义路径（如 `my-secret-login`）
4. 保存配置

## 注意事项

开启后务必记住新登录地址，否则可能无法登录后台。
