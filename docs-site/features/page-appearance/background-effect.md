# 背景特效

::: warning 谨慎
可能影响页面性能。
:::

## 功能说明

为站点添加动态背景效果，提供 8 种样式可选。

## 特效类型

- **星空**：闪烁的星星背景
- **樱花**：飘落的樱花动画
- **收敛线**：粒子连线效果
- **流动线条**：流动的线条动画
- **滴墨水**：墨水扩散效果
- **流动彩带**：彩色丝带飘动
- **随机彩带**：随机彩色粒子
- **质感圆球**：3D 质感圆球

## 开启方法

1. 进入 **魔法工具箱** → **页面外观**
2. 开启「背景特效」
3. 选择特效类型
4. 保存配置

## 注意事项

背景特效会消耗额外的 CPU/GPU 资源，低端设备可能卡顿。建议仅在首页或着陆页开启。
