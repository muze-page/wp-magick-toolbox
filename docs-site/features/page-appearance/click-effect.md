# 点击特效

::: tip 推荐
有趣且安全。
:::

## 功能说明

在页面上点击鼠标时显示特效动画，提供 7 种样式可选。

## 特效类型

- **烟花**：经典烟花爆炸效果
- **文字**：点击时显示随机文字
- **数字**：点击时显示随机数字
- **爱心**：粉色爱心飘散效果
- **星星拖尾**：星星跟随鼠标移动
- **四散烟花**：从点击位置向四周散开
- **圆圈烟花**：圆形扩散效果

## 开启方法

1. 进入 **魔法工具箱** → **页面外观**
2. 开启「点击特效」
3. 选择特效类型
4. 保存配置

## 注意事项

特效仅在桌面端生效，移动端不显示。
