# 文章列表短代码

::: tip 推荐
建议开启。
:::

## 功能说明

使用 `[mabox_single_list]` 短代码在文章或页面中展示文章列表。

## 使用方法

```
[mabox_single_list count="5" category="news"]
```

## 参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| count | 显示文章数量 | 5 |
| category | 指定分类 | 全部 |

## 开启方法

1. 进入 **魔法工具箱** → **短代码**
2. 开启「文章列表短代码」
3. 保存配置
