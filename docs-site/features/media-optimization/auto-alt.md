# 图片自动添加 Alt

::: tip 推荐
安全且建议开启。提升 SEO 和无障碍访问体验。
:::

## 功能说明

WordPress 中上传的图片如果没有手动设置 Alt 文本，前端将缺少无障碍描述。此功能自动使用图片标题作为 Alt 文本。

## 使用场景

- SEO 优化：搜索引擎依赖 Alt 文本理解图片内容
- 无障碍访问：屏幕阅读器需要 Alt 文本来描述图片
- 批量上传：大量图片时手动添加 Alt 效率低

## 开启方法

1. 进入 **魔法工具箱** → **媒体优化**
2. 开启「图片自动添加 Alt」
3. 保存配置

## 注意事项

自动生成的 Alt 文本可能不够精确，重要图片建议手动设置描述性 Alt。
