# 上传文件重命名

::: tip 推荐
安全且建议开启。规范文件命名。
:::

## 功能说明

上传图片或文件时，自动将文件名重命名为指定格式，避免中文文件名带来的兼容性问题。

## 重命名规则

- **MD5**：基于原始文件名生成 MD5 哈希值
- **时间戳**：使用上传时间作为文件名
- **随机文字**：生成随机字符串
- **随机数字**：生成随机数字串

## 开启方法

1. 进入 **魔法工具箱** → **媒体优化**
2. 开启「上传文件重命名」
3. 选择重命名规则
4. 保存配置

## 注意事项

开启后新上传的文件将自动重命名，不影响已上传的文件。
