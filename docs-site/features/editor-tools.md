# 特色样板与站点数据

Npcink Site Toolbox 3.2.0 提供 3 个核心区块样板和 1 个动态区块。它们不需要在插件设置页启用；在文章或页面编辑器的区块插入器中直接使用即可。

## 资源下载卡片

在“区块样板 → Npcink Site Toolbox”中选择“资源下载卡片”。插入后可直接修改资源名称、文件信息、说明和按钮链接。它只由 WordPress 核心群组、段落、标题和按钮组成，插入后不依赖插件运行。

## 文章结论卡片

用于在文章末尾突出核心结论、简短摘要和三个行动要点。插入后仍是普通核心区块，可自由调整内容、颜色和间距。

## 来源与版权说明

用于统一填写资料来源、版权归属和转载要求。建议在发布前逐项替换占位文字，不要把示例内容原样发布。

## 站点数据区块

在区块插入器中搜索“站点数据”或“统计”。区块默认显示已发布文章、已批准评论、分类和用户总数；可在右侧区块设置中修改标题并逐项开关。

统计值在页面渲染时从当前站点读取，不会写入文章内容，也不会向外部服务发送数据。区块和旧版站点统计小工具复用同一份请求级统计快照，页面中出现多个实例时不会重复维护不同的查询逻辑。

## 兼容性

动态区块使用 Block API v3，因此 3.2.0 要求 WordPress 6.3 或更高版本。它的编辑器脚本是随插件发布的可读源码，无需额外构建；三个样板插入后只包含核心区块内容。
