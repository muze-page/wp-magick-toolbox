## 技术架构

- **后端**：PHP 7.0+，WordPress 插件标准架构
- **前端设置页**：React + TypeScript + Vite + Ant Design
- **前端展示**：React + TypeScript + Vite（可选加载）
- **图表**：自研 + ECharts
- **数据存储**：WordPress `wp_options` 表（JSON 格式）
- **通信方式**：WordPress AJAX + REST API

## 安全加固（已完成）

- SQL 注入防护（全部使用 `$wpdb->prepare()`）
- CSRF 防护（nonce 验证）
- XSS 防护（输出转义）
- 权限检查（`current_user_can`）
- 输入清洗（`sanitize_text_field`）
