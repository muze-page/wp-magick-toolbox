一个简单的插件，它增加了添加/修改/删除功能，只需从你的仪表板上的“所有帖子”页面上的特色图片。

描述
当你只想管理帖子缩略图或特色图像时，你没有快速编辑选项。您必须转到编辑页面，更改特色图像，保存，然后转到另一个编辑后页面，重复此过程。

现在有了这个插件，你可以直接从“所有帖子”页面上完成。当你激活插件时，会在那里添加一个名为缩略图的新列。从那里，你可以添加、更改或删除你帖子的特色图片。

安装
1.将插件下载为zip
2.转到插件>添加新内容>上传并上传zip文件。（如果您有问题，请解压缩到一个文件夹，并将其上传到wp根安装的plugins目录中）
3.单击“简易缩略图切换器”下的“激活”

或

1.转到插件>从仪表板添加新内容
2.搜索并安装“简易缩略图切换器”
3.单击“简易缩略图切换器”下的“激活”
